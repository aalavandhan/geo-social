[GET] Geo Social Ranking - /api/gsr

This endpoint returns the top k ranked users users around a given query location q.

Parameter Type Description
lat required Float [latitude] point of query location.
lon required Float [longitude] point of query location.
K optional Integer [Top K-Ranks]. Defaults to 1.
radius optional Float [Number of kms]. Defaults to 4 kms.
data_set optional String [Dataset Name]. Defaults to 'data_GSR_Austin'.
